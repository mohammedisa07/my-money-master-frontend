# Money Master - Personal Finance Tracker

A powerful, modern, and intuitive personal finance management application built with React, Vite, and Tailwind CSS.

## Features

- **Dashboard**: Real-time overview of your financial status.
- **Transactions**: Effortlessly track and manage your income and expenses.
- **Accounts**: Manage multiple accounts and transfers.
- **Visual Analytics**: Interactive charts for spending patterns.
- **Mobile Friendly**: Fully responsive design for on-the-go tracking.

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`

## Tech Stack

- React
- Vite
- Tailwind CSS
- Lucide React
- Framer Motion
- UI Components: Shadcn UI
